# Lemon Shock

レモンだけをテーマにした、2ページ構成の React + Vite サイトです。  
コンセプトは **「レモンを見ただけで酸っぱい気持ちになるサイト」**。

## 特徴

- React Router による 2ページ構成
- レモンを絞る動画をヒーロー背景に使用
- 添付画像・動画を組み込んだビジュアル重視のレイアウト
- 黄色・白・黒を軸にした強いコントラスト
- モバイル表示にも対応

## ページ構成

- `/` : TOPページ
  - 動画背景ヒーロー
  - レモンの刺激を視覚で伝えるカードセクション
  - 2ページ目へ誘導する動画付きセクション
- `/experience` : SOUR ROOMページ
  - 画像と動画を組み合わせた体験ページ
  - 酸っぱさを言葉とビジュアルで演出

## 使用技術

- React
- Vite
- React Router DOM
- CSS

## セットアップ

```bash
npm install
npm run dev
```

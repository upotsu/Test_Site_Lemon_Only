# Test_Site_Lemon_Only


## ■ 概要
<img width="1575" height="832" alt="image" src="https://github.com/user-attachments/assets/f1255e69-c87f-42a7-98cc-25e4ad01c433" />

架空サイト制作チャレンジ第二弾  
「レモンを見ただけで酸っぱい気持ちになる架空サイト」をコンセプトに制作した、視覚体験型の架空Webサイトです。  

動画・画像・テキスト演出を組み合わせることで、実際に味覚を刺激するようなUI/UXを目指しました。
※画像にはpexels.comを使用


## ■ 制作背景
ポートフォリオ制作の一環として、単なる情報提供サイトではなく「体験」を重視したサイトを作成しました。

焼肉サイト制作に続き、今回は真逆のコンセプトである「酸味」にフォーカスし、
視覚だけで感覚に訴えるUI設計を意識しています。

---

## ■ 使用技術
- React (Vite)
- React Router DOM
- CSS
---

## ■ 機能一覧
- フルスクリーン動画背景（Heroセクション）
- ページ遷移（2ページ構成）
- レモン画像による視覚演出
- コンセプトに沿ったテキスト演出
- レスポンシブ対応（簡易）

---

## ■ ページ構成
### TOPページ
- レモンを絞る動画を背景にしたHeroセクション
- キャッチコピー表示
- 画像カードによる視覚訴求

### Experienceページ
- レモンの断面画像によるインパクト演出
- 酸味を想起させるテキスト
- 動画・画像を組み合わせた体験構成

---

## ■ 工夫した点
- 動画とテキストを重ねることで「視覚→感覚」への変換を意識
- 黄色・白・黒をベースにしたシンプルかつ印象的な配色
- 余白を活かしたミニマルデザイン
- 直感的に体験できるUI構成
